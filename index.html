Tailwind CSS - Material Design Colour Palettes

This is a full set of palettes for all Material Design Colours.

You can reference the colours as normal i.e. class="bg-red-500"

For every colour there is also a -contrast meant for text / icons i.e. class="text-red-500-contrast" which is white

There is also three base colours, white, black and transparent

We recommend using PurgeCSS!
